# Kakao-School-Meal-Bot
A package for make kakao-talk school meal bot easily

## Install
```
pip3 install babbot
```

## How to use
```
from babbot import bot, DAEJEON

bot(DAEJEON, 'G100000170') # bot(Location, school_code)
```
